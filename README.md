## Running the RAG Project (Quick Start)
From the project folder:

```bash
cd RAG_Customer_Support_Assistant
npm install
pip install -r requirements.txt
npm run dev
```

- Frontend: `http://127.0.0.1:5173`
- Backend API: `http://127.0.0.1:8787`

For production build and additional usage, see `RAG_Customer_Support_Assistant/README.md`.
